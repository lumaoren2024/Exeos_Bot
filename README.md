## 2、config/config.json格式

```json
[
    {
        "extension_ids": "nodeid1",
        "proxies": "http://127.0.0.1:7890",
        "tokens": "token1"
    },
    {
        "extension_ids": "nodeid2",
        "proxies": "socks5://127.0.0.1:7890",
        "tokens": "token2"
    },
    {
        "extension_ids": "nodeid3",
        "proxies": "protocol://user:pass@ip:port",
        "tokens": "token3"
    }
]
```
